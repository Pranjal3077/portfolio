Hybrid ARIMA+Regression Predictive Model Project

Data Visualization and Prediction using Statistical Methods

Introduction

This project presents a Hybrid Predictive Demand Forecasting System that combines Regression analysis and ARIMA modeling to predict future product demand. The goal is to provide accurate forecasts by leveraging historical sales data, enabling data-driven inventory and sales planning decisions.

Objective

To predict future product demand by combining Regression and ARIMA models, providing actionable insights for inventory management, sales planning, and strategic decision-making.

Methodology

- Collected and cleaned historical sales data using Excel Power Query.
- Performed Regression analysis to identify key factors affecting demand.
- Applied ARIMA modeling to forecast future demand trends.
- Built an interactive Excel dashboard with pivot tables, charts, and slicers for dynamic visualization.
- Used conditional formatting to highlight critical trends and anomalies.

Results

The hybrid forecasting model generated accurate demand predictions and highlighted seasonal trends. The interactive dashboard allowed decision-makers to monitor forecasted vs. actual demand, identify high-demand products, and optimize inventory planning effectively.

Summary

This project demonstrates how a hybrid approach combining Regression and ARIMA models can enhance demand forecasting. The Excel dashboard transformed historical sales data into actionable insights, enabling data-driven inventory management and strategic planning.

Data Source

In this project, I used raw sales data from Kaggle.
