Indian Road Accident Analysis (2014–2022)

Data Visualization using Tableau Public

Introduction

Indian Road Accident Analysis (2014–2022) is a real-world data analytics project based on official Government of India open data. The project focuses on analyzing road accident fatalities across multiple years to identify trends, regional patterns, and severity indicators. Using SQL Server for data cleaning and KPI creation and Tableau Public for visualization, this project demonstrates handling messy, multi-source datasets and converting them into meaningful, policy-relevant insights through ethical data storytelling.

Objective

Analyze year-wise trends in road accident deaths (2014–2022) Identify high-fatality states using geographic analysis Measure state-wise change in road accident deaths (2018 → 2019) Compare railway vs road accident death ratios Build policy-relevant KPIs using SQL

Methodology

- Collected real-world road accident datasets from the Government of India open data portal (data.gov.in).
- Cleaned and standardized messy, multi-year datasets with inconsistent formats using SQL Server.
- Created analytical KPIs and summary views using SQL queries, aggregations, and calculations.
- Combined multiple datasets across different years to enable year-wise and state-wise comparisons.
- Exported processed data to Excel for compatibility with Tableau Public.
- Designed an interactive and ethical data visualization dashboard using Tableau Public, clearly stating data limitations.

Results

The final dashboard revealed clear year-wise trends in road accident fatalities across India, highlighted high-risk states through geographic visualization, and identified significant changes in fatalities over time. Key KPIs enabled comparison across years despite varying data scopes, allowing users to quickly understand severity patterns and regional disparities. The insights generated can support data-driven discussions around road safety awareness and policy planning.

Summary

This project demonstrates the complete analytics workflow using real-world government data — from data cleaning and transformation in SQL Server to storytelling through interactive Tableau dashboards. It strengthened practical skills in handling messy datasets, creating usable KPIs, and delivering insights that provide real value, making it a strong stepping stone in my data analytics journey.
