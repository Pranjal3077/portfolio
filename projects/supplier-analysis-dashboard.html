Supplier Analysis Dashboard

Data Visualization using PowerBI

Introduction

This Supplier Analysis Dashboard project focuses on evaluating supplier performance and profitability using SQL Server (SSMS) and visualization tools. The objective is to derive insights related to supplier efficiency, cost management, order fulfillment, and overall contribution to business profitability. This project helps procurement and supply chain teams make data-driven decisions, optimize supplier relationships, and enhance organizational performance.

Objective

Extract supplier and purchase-related data from the AdventureWorks SQL database.

Create meaningful KPIs that represent supplier performance, profitability, and efficiency.

Perform SQL joins, aggregations, grouping, and advanced calculations to analyze supplier trends.

Build a polished, interactive Excel/Tableau/Power BI dashboard for reporting.

Demonstrate real-world BI skills through SQL and data visualization integration.

Methodology

- Collected and cleaned supplier and purchase data from the AdventureWorks SQL database.
- Performed SQL queries with joins, aggregations, and calculations to derive KPIs.
- Created pivot tables, charts, and interactive dashboards to summarize supplier performance.
- Applied filters, slicers, and conditional formatting to highlight trends and insights.
- Integrated SQL results into Excel/Tableau/Power BI for dynamic visualization and reporting.

Results

The final dashboard provided clear insights into supplier performance, profitability, and order fulfillment efficiency. Decision-makers could identify the most profitable suppliers, monitor underperforming ones, and make data-driven procurement decisions in real-time.

Summary

This project demonstrates how SQL and Excel/Tableau/Power BI can serve as powerful business intelligence tools. The dashboard transformed raw supplier and purchase data into actionable insights, enabling interactive analysis and informed decision-making for procurement and supply chain teams.

Data Sources

Data for this Supplier Analysis Dashboard was sourced from the AdventureWorks SQL database , which provides comprehensive supplier, purchase, and product information for real-world analysis.
